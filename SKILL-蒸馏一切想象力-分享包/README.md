# SKILL · 蒸馏一切想象力 — 分享包

打开 `html/SKILL-蒸馏一切想象力.html` 即可完整浏览整套网页 PPT，无需联网即可正常显示（动效在离线时自动降级）。

## 目录结构

```
SKILL-蒸馏一切想象力-分享包/
├── html/                # 网页 PPT 主文件（直接打开这个）
│   ├── SKILL-蒸馏一切想象力.html
│   └── assets/          # 动效引擎本地副本（离线备用）
├── images/              # PPT 中引用的全部图片
├── js/                  # 图标库本地副本（lucide，离线备用）
└── md/                  # 演讲底稿 / 备用稿
    ├── # Agent Skill：蒸馏一切想象力.md
    └── # Agent Skill：蒸馏一切想象力备用稿.md
```

## 如何打开

双击 `html/SKILL-蒸馏一切想象力.html`，或在浏览器地址栏用 `file://` 打开。

## 说明

- 所有图片、图标库、动效引擎均已本地化，分享整个文件夹即可。
- 字体使用 Google Fonts 在线加载；离线时浏览器会自动回退到系统字体，不影响内容阅读。
- 路径均为相对路径，请保持文件夹内部结构不变。
